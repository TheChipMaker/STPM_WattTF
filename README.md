# STPM_WattTF

An open-source Arduino / PlatformIO library for the STMicroelectronics **STPM3x** energy metering ICs, over SPI.

> **Status:** early development (v0.1.0). Single-channel **STPM32** with current-transformer (CT) sensors is the initial target. Shunt and Rogowski-coil support, and STPM33/34 channels, are planned.

Full documentation, wiring, and usage examples are coming as the library takes shape.
